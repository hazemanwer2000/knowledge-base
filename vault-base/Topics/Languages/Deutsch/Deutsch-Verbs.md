### Regular Verbs
---
#### Form
---
*Z. bsp. spielen.*

| Infinitive | Past Participle | Present Participle |
| ---------- | --------------- | ------------------ |
| *spielen*  | *gespielt*      | *spielend*         |

|             | Present   | Past        |
| ----------- | --------- | ----------- |
| *ich*       | *spiele*  | *spielte*   |
| *du*        | *spielst* | *spieltest* |
| *er/es/sie* | *spielt*  | *spieltet*  |
|             |           |             |
| *wir*       | *spielen* | *spielten*  |
| *ihr*       | *spielt*  | *spieltet*  |
| *sie/Sie*   | *spielen* | *spielten*  |
